SplayTreeGrid

SplayTreeGrid<T> class final

A sparse grid implementation using a SplayTreeMap to store elements.

This implementation is a sparse grid, where elements are stored in row-major order, but with default values omitted. This is memory-efficient for grids with large empty regions, but may be slower than a dense grid for many operations.

Examples

The following grid:

final grid = SplayTreeGrid.fromRows([
  ['a', ' ', 'b'],
  ['c', ' ', ' '],
  [' ', ' ', 'd'],
]);
print(grid.toSparseMap());

Produces the following output:

{0: a, 2: b, 3: c, 8: d}
