EfficientIndexGrid

EfficientIndexGrid<T> mixin

A mixin that augments Grid to indicate it supports efficient index access.

This mixin is used to provide hints to algorithms about the internal layout of the grid, which can be used to optimize performance for certain operations.

To implement this mixin, you must override layoutHint and provide a cooresponding implementation for getByIndexUnchecked. If the layout hint is LayoutHint.private, the behavior is undefined.
