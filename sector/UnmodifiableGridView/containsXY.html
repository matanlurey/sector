Returns true if the grid contains an element at the given x and y.

The x and y coordinates are zero-based, where x is the horizontal axis and y is the vertical axis. The origin (0, 0) is at the top-left corner of the grid.

Examples

final grid = Grid.filled(3, 3, 0);
print(grid.containsXY(1, 1)); // true
print(grid.containsXY(3, 3)); // false

Implementation

@override
bool containsXY(int x, int y) => view.containsXY(x, y);
