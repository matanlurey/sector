Returns an iterable that traverses the grid in a specific order.

The order of the traversal is determined by the provided order, which is a Traversal implementation. If not provided, the default order is determined by the specific implementation of the grid, often row-major.

Examples

final grid = Grid.generate(2, 2, (x, y) => (x, y));
for (final (x, y, element) in grid.traverse(rowMajor())) {
  print(element);
}

The output of the example is:

(0, 0)
(1, 0)
(0, 1)
(1, 1)
